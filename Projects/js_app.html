Pokedemon Pokedex

Project Overview

A responsive web app built using vanilla JavaScript and the PokéAPI, allowing users to browse and explore Pokémon with ease.

Technologies Used

  • Javascript (ES6+)
  • HTML5 & CSS
  • Poke API
  • jQuery - for DOM manipulation
  • Bootstrap for UI components (viaCDN)

Objective

The Goal of the exercise was to build a small web application that loads data from an external API using HTML, CSS and Javascipt so users are able to view a list of Pokemon and see more details for a given Pokemon on demand.

The Experience

This was the first time I’d built a responsive web app using Javascript and accessing information from a public API. It was a great learning experience for me, discovering how to use Javascript functions to deliver required information to the web page, asynchronous behaviour, jQuery, Bootstrap and the joys of debugging!

The process included:

  • Learning about Javascript functions
  • Practicing with the core principles of JavaScript: objects, arrays, conditionals, and loops
  • Adding a complex UI pattern (showing details in a Modal) for the app
  • Styling the UI of the app and implementing the Bootstrap library
  • Implementing accessibility.
