myFlix Angular app

Project Overview

A responsive, single-page Angular application that allows users to explore a curated selection of classic movies. This is the Angular built front-end client for the Movie-Geeks REST API.
Users are able to:

  • View movie data, including details about genres, and directors.
  • Register and manage user accounts
  • Add or remove favorite movies
  • Upload a profile picture

Technologies Used

  • Angular 19
  • Angular Material
  • TypeScript
  • Typedoc
  • RxJS
  • MongoDB, Express.js (back-end API)
  • JWT Authentication (handled by the Movie-Geeks API backend)
  • Github pages for app hosting

Objective

Rebuild the client side to the Movie-Geeks API using Angular instead of React. The user requirements are much the same, users can signup or login to the app, view all the movies, get information about a movies director genre and the movie bio, save a movie to a favourites list in a user profile and change their details.

The Experience

It was great to measure the differences using the Angular framework utilizing TypeScript in a straight comparison to React by designing the same frontend to the same Movie-Geeks API. I really liked the tight structure of Angular components (I used the newer standalone components), and the two way data binding, even though it may seem quite cumbersome with the large bundle size. I also found the use of generics with Typescript and its tight structure very helpful in picking up and pinpointing type errors in the editor. I look forward to learning more about Typescript. I made a few changes with the styling and functionality of the app, using a side scrolling feature to display all films, and dialog boxes to display the features rather than a single movie view but kept to the art deco theme

The process included:

  • Create the app using Angular CLI
  • Implement user registration and login forms using Angular Material
  • Implement Angular components and define the app routing
  • Style the app using the React Bootstrap library
  • Comment the backend and frontend code using JSDoc and TypeDoc
