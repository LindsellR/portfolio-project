Movie-Geeks Server Side Component

Project Overview

A mobile chat application built using React Native, Expo, and Firebase that allows users to sign in and personalise their chat screen, send and receive messages, images and their location data via a map.

Technologies Used

  • Node.js
  • React Native
  • Expo CLI
  • Google Firestore Database
  • JWT Authentication via Passport
  • Gifted Chat Library

Objective

Create a chat app where users can sign in anonymously, send real time messages and images, use location sharing and store messages offline.

The Experience

This was a great exercise designing a cross-platform native app with specific assets(Background image) and to specific design specifications regarding the layout of the welcome screen (font size and weight , container positioning) and specific color choices for the background colors. There were some challenges with compatibility of different versions of technologies, but I find this offers some learning opportunities in finding solutions. Using the android simulator gave less options for debugging than the web browser, which made this more difficult at times, but on the whole another great learning experience.

The process included:

  • Setting up editor for React Native and Expo
  • Creating the layout using native UI components
  • Styling the start screen based on a design brief
  • Build the chat screen and functionality with the Gifted Chat library
  • Authenticate users and store chat messages in Firestore as well as on the device when users are online.
